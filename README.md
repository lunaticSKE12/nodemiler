# Nodemailer

Send mail via node use nodemiler

Create project

`npm init -y`

Install

`npm install --save nodemailer`

use gmail need to “Allow less secure apps” [use this link ](https://www.google.com/settings/security/lesssecureapps)

choose between text or html

Run

`node app.js`
